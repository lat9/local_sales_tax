Prerequisites

This version of the Local Sales Tax requires:

  1. A working copy of Zen Cart 2.1.0 or later.
  2. The site must use a PHP version greater than or equal to PHP 8.0.
  3. (optional) A working copy of Edit Orders, v5.0.0 or later.

Overview

This plugin adds the support for local sales tax zones. In the United States a local sales tax is a additional tax levied by the county or city in addition to the tax levied by a state.

You should consult your local tax authority for tax advice as it relates to your business.

Installation

Follow these steps to install the plugin:

  1. Unzip the plugin's zip-file distribution.
  2. Copy the contents for the distribution's zc_plugins directory to your site's /zc_plugins directory.
  3. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  4. Click on the row that displays a Plugin Key of LocalSalesTax and click the associated Install button.
  5. Next, you'll need to configure your site's local sales taxes; see the Configuring tab for more information.
Uninstalling

To uninstall the plugin:

  1. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  2. Click on the row that displays a Plugin Key of LocalSalesTax and click the associated Un-Install button.
    • That button-click will remove the plugin's configuration settings and "inactivate" the plugin.
    • The database table (tax_rates_local) added during the plugin's installation is not removed when the plugin is uninstalled! To remove that table from your site's database, enter the following SQL in your admin's Tools :: Install SQL Patches:
      DROP TABLE tax_rates_local;
  3. If you also want to remove the files associated with the plugin, click the Clean Up button.

Configuring

The plugin is released with a tool you'll use to configure your local sales taxes and an order-total module that uses those definitions.

First, set up the local sales taxes. Refer to the plugin's wiki article for additional instructions.

Once the site's local sales taxes are set up, navigate to the admin's Modules :: Order Total; you'll see an order-total named Local Sales Tax. Click on the associated Install button, and configure the order-total.
